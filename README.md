# open-api-sdk-php

## О проекте

Данная библиотека предназначена для работы с Open API.

## Требования

* PHP 7.4 и выше
* PHP extension cURL
* Extension Predis

## Установка

```
composer require business-ru/open-api-sdk-php
```

Документация: https://app.swaggerhub.com/apis/Business.Ru/check.business.ru/1.2.2
